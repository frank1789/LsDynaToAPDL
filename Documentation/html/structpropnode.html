Detailed Description

template<typename N, typename P>
struct propnode< N, P >

The propnode struct contains the information of a single node.

The information is divided into: the ID of the node that uniquely identifies it, the x coordinate, the y coordinate, the z coordinate. We use a template definition to be able to create multiple definitions depending on the nodes used. It is suggested to use integer data types to define node IDs, while for float data type coordinates.

Parameters
Ntype for node IDs
Ptype for spatial coordinate
