Detailed Description

template<typename N, typename EN, typename T, int size>
struct propelem< N, EN, T, size >

The propelem struct defines the general appearance of the parameters. of an element.

The content of the structure represents the element defined by a unique ID, by the number of nodes that constitute it. The thickness used for shell-like elements. A template formulation is used to make most of the definitions available to the structure. It is suggested to use integer data types:

  • element IDs;
  • nodes that constitute it as identified by the ID;
  • number of nodes of the element;
    Float Data type:
  • thickness.
    Parameters
    Ntype for elments IDs
    ENtype for node IDs
    Ttype for element's thickness
    sizesets the number of nodes of the element
